# IP Terms Dictionary

Multilingual IP law dictionary (JA/DE/EN/IT) built as a PWA.

## Features

- Japanese / German / English / Italian terms
- Trademark law vocabulary (initial set)
- Simple search
- Offline support (PWA)

## Structure

- `index.html` — main UI
- `cards.json` — term data
- `manifest.webmanifest` — PWA config
- `sw.js` — service worker

## Usage

Open the app in browser:

https://YOUR_USERNAME.github.io/ip-terms-dict/

(Replace YOUR_USERNAME with your GitHub username)

## Notes

This is an experimental learning-oriented dictionary, not a legal reference.
